# XRC721 Token Standard

XRC721 is the non-fungible token (NFT) standard on XDC Network, fully compatible with ERC-721. This guide covers NFT contract creation, metadata hosting, minting, and marketplace integration.

---

## Overview

| Feature | XRC721 | ERC-721 (Ethereum) |
|---------|--------|-------------------|
| Uniqueness | Each token is unique | Each token is unique |
| Metadata | URI-based (IPFS/HTTP) | URI-based (IPFS/HTTP) |
| Gas cost (mint) | ~0.0005 XDC | ~$5–$20 |
| Block time | ~2 seconds | ~12 seconds |

XRC721 tokens are ideal for:
- Digital art and collectibles
- Real estate tokenization
- Identity and certifications
- Gaming assets

## Quick Start — Mint Your First NFT

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.20;

import "@openzeppelin/contracts/token/ERC721/ERC721.sol";
import "@openzeppelin/contracts/access/Ownable.sol";

contract MyNFT is ERC721, Ownable {
    uint256 private _tokenIds;
    string private _baseTokenURI;

    constructor(string memory baseURI) ERC721("MyNFT", "MNFT") Ownable(msg.sender) {
        _baseTokenURI = baseURI;
    }

    function mint(address to) public onlyOwner returns (uint256) {
        _tokenIds++;
        uint256 newTokenId = _tokenIds;
        _safeMint(to, newTokenId);
        return newTokenId;
    }

    function _baseURI() internal view override returns (string memory) {
        return _baseTokenURI;
    }
}
```

## Writing an XRC721 Contract

### Basic Minting Contract

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.20;

import "@openzeppelin/contracts/token/ERC721/ERC721.sol";
import "@openzeppelin/contracts/access/Ownable.sol";
import "@openzeppelin/contracts/utils/ReentrancyGuard.sol";

contract XRC721Token is ERC721, Ownable, ReentrancyGuard {
    uint256 private _tokenIds;
    uint256 public mintPrice = 0.01 ether;
    uint256 public maxSupply = 10000;
    string private _baseTokenURI;
    bool public mintingEnabled = false;

    mapping(uint256 => string) private _tokenURIs;

    event NFTMinted(address indexed minter, uint256 indexed tokenId);

    constructor(
        string memory name,
        string memory symbol,
        string memory baseURI
    ) ERC721(name, symbol) Ownable(msg.sender) {
        _baseTokenURI = baseURI;
    }

    function mint() public payable nonReentrant returns (uint256) {
        require(mintingEnabled, "Minting not enabled");
        require(msg.value >= mintPrice, "Insufficient payment");
        require(_tokenIds < maxSupply, "Max supply reached");

        _tokenIds++;
        uint256 newTokenId = _tokenIds;
        _safeMint(msg.sender, newTokenId);

        emit NFTMinted(msg.sender, newTokenId);
        return newTokenId;
    }

    function mintTo(address to) public onlyOwner returns (uint256) {
        require(_tokenIds < maxSupply, "Max supply reached");

        _tokenIds++;
        uint256 newTokenId = _tokenIds;
        _safeMint(to, newTokenId);

        emit NFTMinted(to, newTokenId);
        return newTokenId;
    }

    function setMintPrice(uint256 newPrice) public onlyOwner {
        mintPrice = newPrice;
    }

    function toggleMinting() public onlyOwner {
        mintingEnabled = !mintingEnabled;
    }

    function setBaseURI(string memory baseURI) public onlyOwner {
        _baseTokenURI = baseURI;
    }

    function withdraw() public onlyOwner {
        (bool success, ) = payable(owner()).call{value: address(this).balance}("");
        require(success, "Transfer failed");
    }

    function _baseURI() internal view override returns (string memory) {
        return _baseTokenURI;
    }

    function tokenURI(uint256 tokenId) public view override returns (string memory) {
        _requireOwned(tokenId);
        return string(abi.encodePacked(_baseURI(), _toString(tokenId), ".json"));
    }

    function totalSupply() public view returns (uint256) {
        return _tokenIds;
    }
}
```

## Metadata and IPFS

### Metadata JSON Format

Each NFT needs a metadata JSON file:

```json
{
  "name": "XDC NFT #1",
  "description": "A unique NFT on XDC Network",
  "image": "ipfs://QmYourImageHash",
  "attributes": [
    {
      "trait_type": "Rarity",
      "value": "Legendary"
    },
    {
      "trait_type": "Power",
      "value": 100,
      "display_type": "number"
    }
  ]
}
```

### Uploading to IPFS

**Using Pinata:**

```bash
# Install Pinata CLI
npm install -g pinata-cli

# Upload image
pinata upload image.png

# Upload metadata JSON
pinata upload metadata/1.json
```

**Using NFT.Storage:**

```javascript
import { NFTStorage, File } from 'nft.storage';

const client = new NFTStorage({ token: API_KEY });

const metadata = await client.store({
  name: 'XDC NFT #1',
  description: 'A unique NFT',
  image: new File([imageBuffer], 'image.png', { type: 'image/png' }),
  attributes: [{ trait_type: 'Rarity', value: 'Legendary' }]
});

console.log(metadata.url); // ipfs://...
```

### Setting Base URI

```solidity
// After uploading all metadata to IPFS
contract.setBaseURI("ipfs://QmYourFolderHash/");
```

Your metadata files should be named `1.json`, `2.json`, etc., matching token IDs.

## Deployment

### Hardhat Script

```typescript
import { ethers } from "hardhat";

async function main() {
  const NFT = await ethers.getContractFactory("XRC721Token");
  const nft = await NFT.deploy(
    "MyNFTCollection",
    "MNFT",
    "ipfs://QmYourFolderHash/"
  );

  await nft.waitForDeployment();
  console.log("NFT deployed to:", await nft.getAddress());

  // Enable minting
  await nft.toggleMinting();
}

main().catch(console.error);
```

### Foundry

```bash
forge create src/MyNFT.sol:XRC721Token \
  --rpc-url https://rpc.apothem.network \
  --private-key $PRIVATE_KEY \
  --constructor-args "MyNFTCollection" "MNFT" "ipfs://QmYourFolderHash/"
```

## Minting Frontend Example

```html
<!DOCTYPE html>
<html>
<head>
  <script src="https://cdn.ethers.io/lib/ethers-5.7.umd.min.js"></script>
</head>
<body>
  <button id="connect">Connect Wallet</button>
  <button id="mint" disabled>Mint NFT (0.01 XDC)</button>

  <script>
    const CONTRACT_ADDRESS = "0x...";
    const ABI = [
      "function mint() payable returns (uint256)",
      "function mintPrice() view returns (uint256)"
    ];

    let provider, signer, contract;

    document.getElementById('connect').onclick = async () => {
      provider = new ethers.providers.Web3Provider(window.ethereum);
      await provider.send("eth_requestAccounts", []);
      signer = provider.getSigner();
      contract = new ethers.Contract(CONTRACT_ADDRESS, ABI, signer);
      document.getElementById('mint').disabled = false;
    };

    document.getElementById('mint').onclick = async () => {
      const price = await contract.mintPrice();
      const tx = await contract.mint({ value: price });
      await tx.wait();
      alert("NFT minted!");
    };
  </script>
</body>
</html>
```

## Security Best Practices

| Practice | Implementation |
|----------|---------------|
| ReentrancyGuard | Use on mint functions |
| Pull over push | Use withdraw pattern for funds |
| Max supply cap | Prevent unlimited minting |
| Access control | Only owner for admin functions |
| URI validation | Validate metadata format |
